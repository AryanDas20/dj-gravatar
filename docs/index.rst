=======================================
Welcome to dj-gravatar's documentation!
=======================================
**Dj-Gravatar** is a Python package to display avatars using `Gravatar`_ on a website developed using Django Framework.

Installation
------------

Windows OS
~~~~~~~~~~
In the terminal/Powershell, type::

   python -m pip install dj-gravatar

If you do not have ``pip``, then download the package from `here`_, unzip it and keep it in your Django project's root directory (as an app).

Linux/MacOS
~~~~~~~~~~~
In the terminal, type::

    sudo pip3 install dj-gravatar

If you are in a virtual environment, type::

    pip install dj-gravatar

If you do not have ``pip``, then download the package from `here`_, unzip it and keep it in your Django project's root directory (as an app).
